# Contributing to NaijaLancers

Thank you for your interest in contributing to NaijaLancers! We welcome contributions from the community.

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Run tests: `npm run lint`
5. Commit your changes: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature`
7. Open a Pull Request

## Development Setup

See the [Quick Start](#quick-start) section in README.md for setup instructions.

## Code Style

- Use TypeScript for type safety
- Follow ESLint rules
- Write descriptive commit messages

## Reporting Issues

Use GitHub Issues to report bugs or request features. Please include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details
