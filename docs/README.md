## Project vision

NaijaLancers is designed to empower Nigerian freelancers and small businesses by providing:

- secure Web3 wallet access
- escrow-backed payment flows
- airtime/data VTU services
- live collaboration through video calls
- AI support for user engagement
- an extensible API and MiniPay integration layer
